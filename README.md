# DOCtoFB2 - Конвертер для Литрес Самиздат

Программа для конвертации файлов DOC/DOCX в формат FB2 с соблюдением правил платформы Литрес Самиздат.

## Особенности

- **Графический интерфейс** на PyQt5
- **Drag-and-drop** загрузка файлов
- **Редактирование FB2** после конвертации
- **Настройки сохранения** файлов
- **Поддержка изображений** и форматирования
- **Валидация FB2** файлов

## Установка

### Вариант 1: Запуск из исходного кода

1. Установите Python 3.8+
2. Установите зависимости:
pip install -r requirements.txt

text
3. Запустите программу:
python doctofb2.py

text

### Вариант 2: Использование готового EXE

1. Скачайте `DOCtoFB2.exe` из релизов
2. Запустите исполняемый файл

### Вариант 3: Сборка из исходников

1. Установите cx_Freeze:
pip install cx_freeze

text
2. Соберите исполняемый файл:
python setup.py build

text
3. Найдите EXE файл в папке `build`

## Использование

1. **Загрузка файла:**
- Нажмите "Загрузить" в меню
- Или перетащите DOC/DOCX файл на окно программы

2. **Конвертация:**
- Нажмите "Конвертировать"
- Результат появится в правой панели

3. **Редактирование:**
- Отредактируйте FB2 файл в правой панели
- Изменения сохраняются при нажатии "Сохранить"

4. **Сохранение:**
- "Сохранить" - сохранить с именем по умолчанию
- "Сохранить как..." - выбрать имя и путь

## Настройки

В меню "Настройки" доступны:
- Путь для сохранения по умолчанию
- Параметры конвертации
- Правила Литрес

## Правила Литрес

Программа учитывает основные требования:
- Заголовки оформляются через стили Word
- Изображения конвертируются в JPG
- Удаляются лишние пустые строки
- Сохраняется основное форматирование

## Системные требования

- Windows 7/8/10/11
- 100 МБ свободного места
- Разрешение экрана 1024x768 минимум

Программа распространяется бесплатно.
